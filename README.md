## What This Is
 
A conversational support agent that helps users reflect on relationship conflicts and generate constructive communication strategies. The agent retrieves relevant therapist-authored responses from a curated knowledge base and synthesizes them into personalized, grounded advice.
 
---
 
## Setup
 
**Requirements:** Python 3.9+
 
**Install dependencies:**
```bash
pip install -r requirements.txt
```
 
**Build the vector index (run once):**
```bash
python rag/build_index.py
```

```bash
streamlit run app/main.py
```

## Knowledge Base
 
**Source:** [nbertagnolli/counsel-chat](https://huggingface.co/datasets/nbertagnolli/counsel-chat) via HuggingFace
 
**What it is:** Real therapist-authored Q&A pairs covering relationship conflict, trust, communication, breakups, and marriage.
 
**What we use:** 225 curated Q&A pairs filtered from ~2,780 raw rows. Filtering criteria:
- Must mention a romantic partner as the core subject
- Answers must be substantive (80+ characters)
- Hard exclusions: crisis content (self-harm, suicide, abuse)
- Two-tier system: Tier 1 (113 high-confidence rows) + Tier 2 (112 extended coverage rows)
 
Each Q&A pair is one document unit embedded as-is — no arbitrary chunking.

## Agent Capabilities
 
### Intent Routing
Classifies every user message into one of:
`rag_qa` | `build_plan` | `reflection` | `save_plan` | `retrieve_plan` | `unsafe` | `out_of_scope`
 
### Knowledge Q&A (RAG)
Retrieves top-3 similar therapist responses from the knowledge base and synthesizes a grounded answer with source attribution.
 
### Actions
 
| Action | Type | Description |
|--------|------|-------------|
| Build Conversation Plan | Multi-turn | Collects issue, goal, tone → generates opening statement, talking points, boundary phrase |
| Reflection Exercise | Single-turn | Generates guided self-reflection prompts based on the user's situation |
| Save / Retrieve Plan | Single-turn | Persists generated plans by session ID for later retrieval |
 
### Guardrails
- Crisis/self-harm detection → redirects to professional resources
- Abuse/violence signals → safe fallback response
- Medical/legal advice requests → out-of-scope rejection
- Prompt injection detection
- Disclaimer: agent is not a licensed therapist
 
### Session Persistence
State is stored in SQLite, including conversation history, filled action slots, and saved plans. State persists across app restarts and can be retrieved by session ID.

## Evaluation
 
15 test cases covering:
- 4 retrieval tests (successful knowledge retrieval)
- 3 routing tests (correct intent classification)
- 3 action tests (multi-turn and single-turn action execution)
- 2 guardrail tests (unsafe content rejection)
- 3 edge case tests (ambiguous input, missing slots, out-of-scope)
 
Run evaluation:
```bash
python evaluation/run_eval.py
```

## Known Limitations
 
- Knowledge base is limited to 225 documents — niche relationship scenarios may not retrieve strong matches
- Agent does not provide professional mental health advice and is not a substitute for therapy
- Session state is local only — no multi-device sync
- LLM responses are grounded in retrieved content but may still reflect biases present in the CounselChat dataset
